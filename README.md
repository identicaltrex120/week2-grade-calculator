*Student Grade Calculator*

Author: Krish Sunil

**Project Description**

The Student Grade Calculator is a Python program that collects marks for multiple students, calculates their average score, assigns grades, and displays performance statistics.
This project demonstrates the use of control flow, data structures, and error handling in Python.

*The program allows users to:*
Enter student details
Calculate grades automatically
View formatted results
Display class statistics
Search for a specific student
Save results to a file
Features
Grade calculation using if/elif/else
Store student data using lists
Input validation using while loops
Error handling using try-except
Calculate average marks
Display highest and lowest scores
Search for student records
Save results to a text file
Menu-driven program
Formatted output table
Color-coded grades

**Technologies Used:**
Python 3
Lists
Loops (for, while)
Functions
Exception Handling
